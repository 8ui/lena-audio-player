# Markers (bookmarks on waveform) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Пользователь ставит/удаляет/навигирует именованные закладки (маркеры) на движущейся волне.

**Architecture:** Чистая marker-математика (sort/relabel/nearest/prev-next) — новый модуль `src/waveform/markers.ts`, покрыт unit-тестами (паттерн `viewport.ts`/`position.ts`). Стор-экшены тонкие: зовут чистые функции + `persist()` + `seek()`. Рендер тиков — императивный, в rAF-цикле `WaveformCanvas` (проверяется вручную/сборкой, как политика canvas). UI — `MarkersControl` по паттерну `LoopControls` (`useShallow`).

**Tech Stack:** React 19, TS 7 (tsgo), zustand 5, Vitest 4 + RTL. Без новых зависимостей.

## Global Constraints

- Только мобильные (portrait). Тач-таргеты ≥ 44×44px (`--touch: 48px`).
- Пур-логика — тест-первый. Императивный canvas — сборка + ручная проверка.
- zustand v5: мульти-полевой селектор → `useShallow` (иначе infinite re-render).
- `Marker { id: string; time: number; label: string }` — уже в `src/types.ts`, не менять.
- `persist()` дебаунсится 400мс; `flushPersist()` на nav-границах.
- Диапазон времени маркера: `0..duration` (кладём на текущей `position`, она уже clamped).

import type { Marker } from '../types';

// A "− маркер" tap removes the marker nearest the playhead, but only if it is
// within this many seconds — so a stray tap far from any marker does nothing.
export const MARKER_DELETE_THRESHOLD = 1;

export function relabel(markers: Marker[]): Marker[] {
  return [...markers]
    .sort((a, b) => a.time - b.time)
    .map((m, i) => ({ ...m, label: String(i + 1) }));
}

export function insertMarker(markers: Marker[], marker: Marker): Marker[] {
  return relabel([...markers, marker]);
}

export function removeNearestMarker(
  markers: Marker[],
  time: number,
  threshold: number = MARKER_DELETE_THRESHOLD,
): Marker[] {
  if (markers.length === 0) return markers;
  let bestIdx = -1;
  let bestDist = Infinity;
  for (let i = 0; i < markers.length; i++) {
    const d = Math.abs(markers[i].time - time);
    if (d < bestDist) {
      bestDist = d;
      bestIdx = i;
    }
  }
  if (bestIdx === -1 || bestDist > threshold) return markers;
  return relabel(markers.filter((_, i) => i !== bestIdx));
}

export function nextMarkerTime(markers: Marker[], time: number): number | null {
  let best: number | null = null;
  for (const m of markers) {
    if (m.time > time && (best === null || m.time < best)) best = m.time;
  }
  return best;
}

export function prevMarkerTime(markers: Marker[], time: number): number | null {
  let best: number | null = null;
  for (const m of markers) {
    if (m.time < time && (best === null || m.time > best)) best = m.time;
  }
  return best;
}
```

import { usePlayerStore } from '../store/usePlayerStore';
import { useShallow } from 'zustand/react/shallow';

export function MarkersControl() {
  // zustand v5: a fresh-object selector needs useShallow (see LoopControls).
  const { markers, addMarker, removeMarker, seekPrevMarker, seekNextMarker } = usePlayerStore(
    useShallow((s) => ({
      markers: s.markers,
      addMarker: s.addMarker,
      removeMarker: s.removeMarker,
      seekPrevMarker: s.seekPrevMarker,
      seekNextMarker: s.seekNextMarker,
    }))
  );
  const has = markers.length > 0;
  return (
    <div className="control markers">
      <button onClick={addMarker}>＋ маркер</button>
      <button aria-label="предыдущий маркер" onClick={seekPrevMarker} disabled={!has}>◀</button>
      <button aria-label="следующий маркер" onClick={seekNextMarker} disabled={!has}>▶</button>
      <button aria-label="удалить маркер" onClick={removeMarker} disabled={!has}>− маркер</button>
    </div>
  );
}
```
